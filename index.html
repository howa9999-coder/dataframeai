Pandas DataFrame Functions â€“ Explanations & Examples

This page contains the most commonly used functions and commands in pandas for working with DataFrames. Each function includes an explanation and a working example.

1. Importing pandas

import pandas as pd

2. Creating a DataFrame

Explanation: Create a DataFrame from a dictionary or list.

data = {'Name': ['Alice', 'Bob'], 'Age': [25, 30]}
df = pd.DataFrame(data)
print(df)

3. df.head(n)

Explanation: Returns the first n rows. Default is 5.

df.head()

4. df.tail(n)

Explanation: Returns the last n rows.

df.tail()

5. df.info()

Explanation: Displays summary including data types, non-null values.

df.info()

6. df.describe()

Explanation: Generates summary statistics for numerical columns.

df.describe()

7. df.columns

Explanation: Lists all column names.

print(df.columns)

8. df.shape

Explanation: Returns the dimensions (rows, columns).

print(df.shape)

9. df.dtypes

Explanation: Shows data type of each column.

print(df.dtypes)

10. df['column'] or df.column

Explanation: Access a column.

print(df['Name'])
print(df.Name)

11. df.loc[row_label]

Explanation: Access rows and columns by label.

df.loc[0]  # First row

12. df.iloc[row_index]

Explanation: Access rows and columns by index.

df.iloc[1]  # Second row

13. df.set_index('column')

Explanation: Set a column as the row index.

df.set_index('Name')

14. df.reset_index()

Explanation: Reset index to default integer index.

df.reset_index()

15. df.sort_values(by='column')

Explanation: Sort DataFrame by a specific column.

df.sort_values(by='Age')

16. df.isnull()

Explanation: Returns DataFrame showing True for null values.

df.isnull()

17. df.dropna()

Explanation: Removes rows with missing values.

df.dropna()

18. df.fillna(value)

Explanation: Fill missing values with a specified value.

df.fillna(0)

19. df.drop('column', axis=1)

Explanation: Drop a column from the DataFrame.

df.drop('Age', axis=1)

20. df.apply(function)

Explanation: Apply a function to each column or row.

df['Age'].apply(lambda x: x + 1)

21. df.groupby('column')

Explanation: Group data based on a column.

df.groupby('Name').mean()

22. df.merge(df2, on='column')

Explanation: Merge two DataFrames on a common column.

pd.merge(df1, df2, on='ID')

23. pd.concat([df1, df2])

Explanation: Concatenate two DataFrames.

pd.concat([df1, df2])

24. df.to_csv('file.csv')

Explanation: Save DataFrame to a CSV file.

df.to_csv('data.csv', index=False)

25. pd.read_csv('file.csv')

Explanation: Load a CSV file into a DataFrame.

df = pd.read_csv('data.csv')

26. df.value_counts()

Explanation: Count unique values in a Series.

df['Name'].value_counts()

27. df.duplicated()

Explanation: Detect duplicate rows.

df.duplicated()

28. df.drop_duplicates()

Explanation: Remove duplicate rows.

df.drop_duplicates()

29. df.rename(columns={'old': 'new'})

Explanation: Rename column names.

df.rename(columns={'Name': 'Full Name'})

30. df.query("condition")

Explanation: Query DataFrame using a string expression.

df.query("Age > 25")

Note: This is not an exhaustive list, but it covers the most commonly used functions for working with pandas DataFrames.
